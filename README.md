# Genius Moves Landing Page

A responsive static landing page for GitHub Pages.

## Files

- `index.html` — main page
- `styles.css` — all page styling
- `script.js` — mobile navigation and dynamic copyright year
- `app-ads.txt` — AdMob verification file

## Publish on GitHub Pages

1. Create a public repository named exactly `developershuaib.github.io`.
2. Upload all four files to the root of that repository.
3. Open **Settings → Pages** in the repository.
4. Under **Build and deployment**, select:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
5. Save.
6. Confirm these addresses work:
   - `https://developershuaib.github.io/`
   - `https://developershuaib.github.io/app-ads.txt`

## Before publishing

Replace the placeholder Magneterium Google Play link (`href="#"`) in `index.html`
with the actual Play Store URL.

The contact email currently used is `developershuaib@gmail.com`.
Change it if needed.
